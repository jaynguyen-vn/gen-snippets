KEY PATTERNS
============

1. SINGLETON SERVICES
   - TextReplacementService, LocalStorageService, AccessibilityPermissionManager
   - Static .shared property for app-wide access
   - Thread-safe with DispatchQueue + NSLock

2. OBSERVABLE STATE MANAGEMENT
   - @Published variables in ViewModels
   - Combine publishers for reactive updates
   - Single source of truth via LocalStorageService

3. THREAD SAFETY MODEL
   - DispatchQueue(concurrent) for reader-writer patterns
   - os_unfair_lock for callback fast path performance
   - NSLock within suffix tree nodes

4. DATA FLOW
   UserDefaults → LocalStorageService (cached/batched)
      ↓
   {LocalSnippetsViewModel, CategoryViewModel}
      ↓
   Views (ThreeColumnView, ModernSnippetSearchView)
      ↓
   TextReplacementService (keyboard monitoring → replacement)

CORE FEATURES
=============

TEXT REPLACEMENT:
  • CGEvent tap for system-wide keyboard monitoring
  • TrieNode suffix tree for O(m) command matching
  • Rolling buffer: last 50 characters, 15-second timeout
  • Thread-safe per-app customization via EdgeCaseHandler

DYNAMIC KEYWORDS:
  {clipboard}, {cursor}, {timestamp}, {random-number}
  {dd/mm}, {dd/mm/yyyy}, {time}, {hh:mm}
  {uuid}, {weekday}, {month}

RICH CONTENT:
  • File-based storage: ~/Library/Application Support/GenSnippets/RichContent/
  • Supports: images (PNG), files, URLs
  • Base64 fallback for legacy compatibility
  • Multi-file per snippet support

IMPORT/EXPORT:
  • Share snippets/categories as JSON
  • Conflict detection on import
  • Skip/overwrite/rename conflict resolutions
  • Rich content included in exports (as Base64 for portability)

APP-SPECIFIC CUSTOMIZATION (EdgeCaseHandler):
  Browser, Terminal, Discord, IDE, VM, RemoteDesktop, PasswordField, Game
  Per-app: deletion delay, paste delay, simple deletion, expansion disable flags
